# Focused on the User, Overlooking the Risks: Security and Privacy Understandings, Practices and Challenges of Independent Chinese AI Agent Developers
This repository hosts the artifact for our paper, "Focused on the User, Overlooking the Risks: Security and Privacy Understandings, Practices and Challenges of Independent Chinese AI Agent Developers", detailed information.

The interview script contains an interview outline for discussions with individual AI Agent developers, primarily covering four sections: General Understanding, Development Practices and Tools, Probe/Security Scenarios, and Expectations.

The codebook for the data analysis includes themes for three research questions, along with their subthemes and descriptions.
